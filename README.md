## 产品 PRD 设计

### 核心功能点
1. **用户登录与身份验证** - 手机号+验证码/密码登录、Token管理、身份状态保持
2. **实时定位打卡** - GPS定位获取、Wi-Fi定位验证、上下班打卡、打卡范围校验
3. **外勤打卡** - 外勤原因填写、拍照上传、位置确认
4. **请假申请** - 请假类型选择、时间范围设置、原因说明、审批状态追踪
5. **打卡记录查询** - 日历视图、列表视图、异常标记、详情查看
6. **统计分析** - 出勤率统计、迟到早退次数、月度考勤报表、可视化图表
7. **个人中心** - 个人信息管理、密码修改、帮助中心、设置
